# 项目简介

基于pyqt的家庭鼠患智能检测系统，是一款结合深度学习技术的应用。其主要用途在于通过部署在家中的摄像头，自动检测并识别鼠类活动，以助于及时采取防治措施。系统功能模块包括图像采集、实时检测、数据记录及报警提示。在技术特点上，本系统采用了基于pyqt的图形用户界面，确保操作简便；同时，深度学习算法使检测准确性得到有效提升，适用于家庭环境中的鼠患管理。


## 核心功能
- 图像采集：通过部署在家中的摄像头捕获实时画面
- 实时检测：自动识别并分析视频流中的鼠类活动
- 数据记录：保存检测到的鼠患相关信息
- 报警提示：在发现鼠患活动时提供及时提醒

## 适用场景
- 家庭环境的鼠患监测与管理
- 住宅区域的鼠患早期预警
- 家庭安防系统的辅助功能

## 技术特点
- 基于PyQt的图形用户界面，操作简便直观
- 集成深度学习算法，提高鼠患检测的准确性
- 专为家庭环境设计，适应常见的家居监控环境
